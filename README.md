## 🌟 Overview

The **SECE Intelligent RAG Assistant** is a comprehensive solution designed to fetch, process, and intuitively answer questions based on the official Sri Eshwar College of Engineering website. It bridges the gap between raw web data and natural language queries, empowering users to find relevant college information instantly.

The project features a powerful web scraper, an advanced embedding-based semantic search engine, and two interfaces: a sleek **Web App** (Flask) and a blazing-fast **CLI Assistant**.

## ✨ Key Features

- **🕸️ Automated Web Scraper (`Scarper.py`)**: Dynamically extracts paragraphs, headings, navigation links, and metadata from the SECE website using `BeautifulSoup4`.
- **🧠 Semantic Search Engine**: Utilizes Hugging Face's `sentence-transformers` (`all-MiniLM-L6-v2`) to compute high-dimensional embeddings of text chunks.
- **⚡ High-Speed Retrieval**: Implements Scikit-Learn's Cosine Similarity to retrieve the top matching information accurately in milliseconds.
- **💬 Interactive Web UI (`app.py`)**: A modern Flask-based web interface to chat with the bot natively in your browser.
- **💻 CLI Chatbot (`chatbot.py`)**: A lightweight, terminal-based chatbot for quick testing and local interaction.

## 🚀 Getting Started

### 1. Prerequisites
Make sure you have Python 3.8+ installed.

### 2. Installation
Clone the repository and install the required dependencies:
```bash
git clone https://github.com/prabu411/SECE-Intelligent-RAG-Assistant.git
cd SECE-Intelligent-RAG-Assistant

# Install dependencies
pip install flask requests beautifulsoup4 sentence-transformers scikit-learn numpy
```

### 3. Generate Knowledge Base
Before starting the chatbot, run the scraper to build the localized database:
```bash
python Scarper.py
```
*(This will generate `sece_full.json` and associated CSV files containing the scraped data).*

### 4. Run the Web Application
Launch the Flask server to interact with the bot through a browser:
```bash
python app.py
```
**Navigate to:** `http://localhost:5000`

### 5. Run the CLI Assistant
Prefer the terminal? Run the CLI version directly:
```bash
python chatbot.py
```
